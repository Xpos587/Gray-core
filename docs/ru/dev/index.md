## Протоколы

### VLESS

VLESS — лёгкий протокол без состояния. Основан на формате UUID.

### VMess

VMess — оригинальный протокол V2Ray с улучшенной производительностью.

### REALITY

REALITY — протокол для имитации HTTPS соединений с реальными сайтами.
