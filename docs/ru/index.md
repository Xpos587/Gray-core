# Документация Gray-core

**⚠️ Важно: Образовательные цели**

Gray-core — это **образовательный и исследовательский проект**. Весь код и документация предназначены исключительно для:
- Академического изучения сетевых протоколов
- Исследования сетевого трафика и DPI систем
- Документации технических возможностей
- Научных исследований в области сетевых технологий

**Пользователь несёт ответственность за соблюдение всех применимых законов.**

## О проекте

**Gray-core** — сетевая прокси-платформа для образовательных исследований в области:
- Анализа сетевых протоколов
- Изучения DPI (Deep Packet Inspection) систем
- Документации инфраструктуры цензуры
- Исследования сетевого трафика
